     LittleWigs v39     
-------------------------------------------------------------------------------------
Общее

Авточек РИО      - Автоматическая проверка РИО (.ch), если кто-то пишет вам в личку (есть возможность включения этой функции только при активном ЗСГ). Отображается сокращённая информация (ник и РИО) не более одного анонса для каждого отдельного юнита (после старта/прохождения ключа/релога блэклист для анонса РИО очищается). Также есть возможность включить отображение полной информации при автоматическом использовании команды (.ch). Найти переключатели можно в общем модуле "PARTY INFO" в разделе рейда "Анторус, Пылающий Трон".

АвтоЗСГ          - При линке ключа в общий чат будет автоматически заполняться ЗСГ по данным ключа. Отключить можно в общем модуле "PARTY INFO" в разделе рейда "Анторус, Пылающий Трон".

Запуск ключа     - В фрейм запуска ключа добавлена кнопка для проверки готовности и кнопка для запуска пятисекундного отсчёта.

Таймеры          - Для особо важных кастов реализовано более подробное отображение таймеров (таймер на каждый отдельный каст с отображением метки моба, который его запустил. Если на мобе не было (или не удалось определить) метки в момент запуска таймера, то к названию таймеру будет присвоен номер каста). Реализовано для кастов: "Щит бури", "Разрывающий хлыст Бездны", "Некротический взрыв", "Высвобождение ярости", "Око урагана", "Фазовый прорыв", "Буря глаз", "Командирский крик", "Каменный взгляд", "Бронераковина", "Превращение: рыба", "Энтропийный туман", "Обжигающий дождь", "Леденящий душу вопль", "Буйный вопль", "Разрыв пространства".

Автометки        - Добавлены мобы для выбора в модулях Automarks: "Ночнорожденный-возродитель" (Катакомбы Сурамара), "Лишённый души защитник" (КЧЛ).

Квартал звёзд
Общее            - Небольшие корректировки функции анонса всех баффов через макрос (обнова 38) для более корректного отображения у игроков с повышенным пингом. Исправлены макросы для EN-версии клиента (см. ниже).
Поиск шпиона     - Для ID шпионов реализовано автоматическое сканирование вашей цели серверной командой "_scan". Добавлены подсказки от аддона для использования этой функции (таймер с КД на следующее сканирование, анонс для настоящего/не настоящего шпиона в таргете, анонс в чат, если в таргете настоящий шпион). P.S. для анонса настоящего/не настоящего шпиона при выборе таргета также учитываются результаты сканирования других игроков с этой версией аддона. Если до сих пор кто-то не знает/не понял, как это работает, то просто меняйте таргет с возможным шпионом, когда аддон просит вас это сделать, и ждите несколько секунд для следующего анонса (разговаривать с ними не нужно во время поиска).
Комната активна  - Объявление в чат от лидера или танка при активации первой контрольной точки (комнаты), если в момент активации были игроки вне подземелья.

Престол Триумвирата
Треш             - Таймер для способности "Энтропический туман".

Собор Вечной Ночи
Треш             - Таймер для способности "Обжигающий дождь".

Вечное цветение
Йалну            - Корректировка таймера активации босса. Подсчёт растоптанных цветков.

Чертоги доблести
Треш             - На мобах Олмир Просвящённый и Селстен, которые перед Хирьей, больше не будут появляться лишние (боссовские) таймеры/спеллы.
Четыре короля    - В окно госипки для РЛа добавлена кнопка проверки готовности и кнопка для отправки в чат команды !pull.

Каражан: Верх
Смотритель       - Корректировка таймера "Призыв нестабильной энергии". Таймер+анонс для "Разряд энергии". Удалён анонс стаков баффа у сферы и таймер на прыжок сферы.

Логово Нелтариона
Уларогг Скаломол - Корректировка таймеров.
Даргрул          - Корректировка таймеров/анонсов.
Треш             - Таймер+анонс для способности "Призыв червя".

Утроба Душ
Скьял            - Анонс таргетов/анонс, если на тебе для способности "Не щадить никого".

Крепость Черной Ладьи
Треш             - Корректировка резкости срабатывания анонса для способности "Выстрел" у Восставшей лучницы.
